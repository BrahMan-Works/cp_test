# cp-test

A lightweight competitive programming runner for C++.

## Features

- Compile C++ automatically
- Run with provided input
- Compare output with expected output
- Show ACCEPTED / WRONG ANSWER
- Measure execution time

## Usage

Create a test folder:


.cp-test/input.txt
.cp-test/output.txt


Run the tester:


./cp_test main.cpp


Example output:


ACCEPTED
Execution time: 1.88 ms


## Requirements

- g++
- Linux / Unix environment
